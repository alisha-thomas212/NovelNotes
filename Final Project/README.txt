**NovelNotes** is a simple web app for users to register, log in, and create, store, and manage personal literary notes. This README includes complete usage instructions for both regular users and administrators.

---

### Features

- Register and sign in as a user
- Create reviews and read other reviews for admin and 'guests'
- Admin access to view all users
- SQLite database integration for data storage
- Manual logout via cache clearing (temporary workaround)

---

### INSTALL COMMAND:
To install npm modules execute:
    npm install


### LAUNCH COMMAND:
node server.js
